SpringBoot+Mybatis+Druid批量更新 multi-statement not allow异常

发表于 2023-10-26 分类于 SpringBoot 本文字数： 4.5k 阅读时长 ≈ 4 分钟

  • 问题原因

    • 方案一
    • 方案二

项目环境

SpringBoot: 2.2.6.RELEASE
SpringCloud: 2021.0.5
Druid: 1.1.22
Mybatis-plus: 3.3.1

多条SQL语句同时执行更新操作，提示报错

问题原因

mysql默认不支持批量更新
Druid的multiStatementAllow默认是false

解决办法

方案一

mysql连接url参数添加allowMultiQueries=true
如果配置了druid 注释掉 filters: wall，加上
filter:
wall:
config:
multi-statement-allow: true
none-base-statement-allow: true

方案二

DataSourceProperties配置类添加SQL多条更新支持
