移除默认安装的插件

npm uninstall hexo-generator-index --save

安装新插件


npm install hexo-generator-index-pin-top --save

置顶操作

文章开头分割部分添加

top: true

多篇文章置顶，控制顺序，数值较大的排在前面

top:n

设置置顶图标

打开/themes/next/layout/_macro/post.swig文件，在

下方，插入如下代码：

{% if post.top %}
    
    置顶
    |
{% endif %}
